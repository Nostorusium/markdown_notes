## 图像识别问题

天地图与采样点的选择考虑以下因素。

| 类型 | 表现 | 原因 |
|------|------|------|
| **分辨率低** | 像素化、模糊、细节丢失 | 卫星传感器性能限制 |
| **云雾遮挡** | 部分或全部被云层覆盖 | 拍摄时天气不佳 |
| **阴影严重** | 山体/建筑阴影覆盖 | 太阳高度角低、地形起伏 |
| **对比度差** | 地物区分不明显 | 光照条件、传感器设置 |
| **色彩失真** | 颜色不自然 | 大气散射、传感器校准 |
| **季节不利** | 植被枯萎、雪覆盖 | 冬季影像 |
| **过曝/欠曝** | 太亮或太暗 | 曝光设置问题 |


## 划分策略

划分一块parcel的主要原则是其是否**在功能上是独立地块**。

### 线性分界

| 类型 | 特征描述 |
|-----|---------|
| 田埂 Bund/Ridge  | 凸起的土堆或石砌结构，用于行走或区域分界 |
| 沟渠 Ditch/Canal | 凹陷，可能用于排水或灌溉 |
| 道路 Road        | 地块硬性分界 |
| 围栏/篱笆 Fence  | 明显的线性结构 |

### 面状障碍物

| 类型 | 特征描述 |
|----- |----------|
| 水体 | 池塘、水库、河流等水域 |
| 建筑物 | 农舍、仓库、棚房等建筑 |
| 林地 | 防护林带、果树林等植被 |
| 裸岩/陡坡 | 灰白色，无植被的不可耕作区域，含盐碱地 |

### 弱边界特征

| 类型 | 特征描述 |
|----- |----------|
|色调差异| 可能由不同作物类型、不同生长阶段、不同工作状态造成。


### 可用策略

- 对于层级结构密集，边界狭窄(小于0.5m)的情况，采用共同边界标注。优先对大型连续地块矢量化，通过内线切割为子地块。
- 对于大于等于0.5m的田埂宽度，采取双边切割，沿两侧独立生成边界，去除田埂本身。
